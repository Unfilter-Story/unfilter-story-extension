# Live User Site Guidelines — Unfilter Story
# Editorial Elegance & Grid Brutalism

**Version:** 1.0  
**Target:** Public Facing Website  
**Design Philosophy:** High-impact storytelling, reading-first accessibility, and sharp grid structures.

---

## 1. Color System (Live Site)
The public site uses a sophisticated "Midnight News" palette to create a premium, immersive reading environment.

| Token | Hex Code | Usage |
| :--- | :--- | :--- |
| **Midnight Navy** | `#1A1A2E` | Deep backgrounds and header bars |
| **Pure White** | `#FFFFFF` | Main typography and emphasis |
| **Neutral Gray** | `#D1D5DB` | Secondary links and metadata |
| **Brand Accent** | `#E94560` | "Submit Story" button and category tags |
| **Border Dark** | `#1F2937` | Grid line separators in dark sections |

---

## 2. Typography & Brand Identity
*   **The Logo**: **Playfair Display**, Black (900), Serif. Represents tradition and "The Truth."
*   **Headlines**: **Inter**, Extremely Bold, Sharp tracking.
*   **Body Content**: **Inter / IBM Plex Sans** for neutral, bias-free reading.
*   **Navigation**: Semi-bold, sans-serif, uppercase with subtle tracking.

---

## 3. Component Standards
### 3.1 Content Layout
*   **Brutalist Grid**: 1px solid borders separating content streams.
*   **Aspect Ratios**: 21:9 for hero headers; 4:3 or 1:1 for feed thumbnails.

### 3.2 Navigation & interactivity
*   **Dynamic Dropdowns**: Smooth 300ms transitions with scale-in and opacity fade.
*   **Hover States**: Color shift from Gray to White; underline transitions for links.
*   **Mobile Experience**: Condensed list menus with high-tap-target areas.

---

## 4. Visual Integrity
*   **Zero Shadows**: Depth is created through stacking and transparency, not drop-shadows.
*   **Compression**: High-contrast imagery used as backdrops with 20-30% dark overlays for legibility.
